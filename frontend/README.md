# Smart AI Advisory System - Frontend

React frontend for the Smart AI Advisory System for Smallholder Farmers.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at http://localhost:3000

## Features

- User authentication (Login/Register)
- Dashboard with weather snapshot and quick actions
- AI Chat Advisory
- Weather Intelligence
- Plant Disease Detection
- Market Prices with charts and predictions
- Seasonal Planting Guide

## Tech Stack

- React 18
- React Router DOM
- Tailwind CSS
- Axios
- Recharts
- Lucide React Icons
